# Khanqah Maseehul Ummat - Website

## 📁 Project Structure
```
khanqah/
├── index.html          ← Main website file (open this in browser)
├── css/
│   └── style.css       ← All styles
├── js/
│   └── app.js          ← All functionality
└── images/             ← ⭐ PUT YOUR IMAGES HERE
    ├── logo.png         ← Khanqah logo (auto-detected by name)
    ├── hero.jpg         ← Optional hero background image
    ├── banner.jpg       ← Optional alternate banner
    └── favicon.png      ← Browser tab icon
```

## 🖼️ How Images Work
Just place your images in the `images/` folder with these names:
- `logo.png` → Appears in navbar and favicon
- `hero.jpg` or `banner.jpg` → Hero section background
- Any other images → Available for use

## 🔧 Features
- **Home** - Welcome page with stats
- **Notifications** - Announcements and events
- **Courses** - 6 Islamic courses listed
- **Audio Bayans** - Admin can add audio links; auto-detects title/category/duration
- **Video Gallery** - Paste YouTube links; watch on site or YouTube
- **Q&A** - AI auto-categorizes questions using Claude API
- **Connect** - Facebook, Instagram, YouTube, WhatsApp

## 🚀 How to Use
1. Download the `khanqah` folder
2. Place your images in the `images/` folder
3. Update social media links in `index.html`
4. Open `index.html` in any web browser
5. For hosting, upload the entire folder to any web host

## 🔒 Admin Features
- **Audio Admin Panel**: Add bayan URL → auto-fetches info → adds to library
- **Video Admin Panel**: Paste YouTube URL → shows in gallery with thumbnail
- **Q&A Auto-Category**: AI detects question category automatically
